# Données

Ce dossier contient les données locales non versionnées.

- `raw/` : fichiers bruts Agreste, CLC et communes IGN.
- `processed/` : bases reconstruites par les scripts, dont `twfe_data`.


Les données brutes ne sont pas stockées dans le dépôt GitHub en raison de leur taille.
Elles sont archivées sur Zenodo et récupérées à l'aide du script 0_data_telechargement.R
